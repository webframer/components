# Code Example

A UI view to display live code example, along with source code used for generating that example.

Webframe documentation uses `CodeExample` to demo component usage, such as [this](/docs/ui/components/View#examples).

All examples have a layout switch to toggle between `LTR` (Left to Right) and `RTL` (Right to Left) directions for testing.
