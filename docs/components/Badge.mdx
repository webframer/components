# Badge

<AssetCard className='float-right margin' component={Badge} />

An element to display shortened numbers - usually an alert or notification count on top of menu icons.

The count number is shortened to `maxDigits` length by rounding to the last retained digit. Additionally, it may have a corresponding letter appended.
