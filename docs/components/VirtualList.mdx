# VirtualList

A [View](View) container component to display large list of items without performance penalty.

### Unique Features

1. Items can vary in sizes (both width and height)
2. No need to specify or calculate item dimensions
3. Grid layout support.

Virtual List works by only rendering the visible items in the list that are in view, and the surrounding items within `renderRadius`.
