# Tooltip

Tooltips display additional information (text, images, or explanations), when users hover over, focus on, click or tap an element.

## Unique Features

### 1. Intuitive Code Markup

Webframe Tooltip inserts itself as a child of any component that needs popover, instead of wrapping the element like most UI libraries. The benefits of this is that you can have:
- Multiple tooltips for the same element without JSX nesting nightmare
- Markup that aligns with human thinking, see [code examples](#examples) below.

### 2. Render Modes

I. `position: fixed` - by default, Tooltip renders inside a React Portal, as direct child of the document body, to ensure correct positioning for most use cases.
II. `position: absolute` - Tooltip with `embedded`  set to true renders inside the parent node with correct event bubbling and keyboard accessibility. This is great for UX, like nested dropdown menus.

### 3. Weights 1kB gzipped

This is in contrast to similar components using react-popper, like Material UI Popper, which weights 24.9kB gzipped (at the moment of this writing).

## Examples

<CodeExample desc='Default Tooltip'>
  <Button>
    <Text>Hover or Focus</Text>
    <Tooltip>Tooltip</Tooltip>
  </Button>
</CodeExample>

<CodeExample desc='As `tooltip` property of `Button`'>
  <Button tooltip='Tooltip'>
    <Text>Hover or Focus</Text>
  </Button>
</CodeExample>

<CodeExample desc='Open or close on click'>
  <Button>
    <Text>Click to Open</Text>
    <Tooltip on='click'>Tooltip</Tooltip>
  </Button>
</CodeExample>

<CodeExample desc='Tooltip Position'>
  <Button>
    <Text>Click to Show Positions</Text>
    <Tooltip on='click' position='top'>Top</Tooltip>
    <Tooltip on='click' position='right'>Right</Tooltip>
    <Tooltip on='click' position='bottom'>Bottom</Tooltip>
    <Tooltip on='click' position='left'>Left</Tooltip>
  </Button>
</CodeExample>

<CodeExample desc='Tooltip Alignment'>
  <Button>
    <View>
      <Text className='padding'>Click to Show Position + Align</Text>
      <Text className='padding'>Tooltip Component Options</Text>
    </View>
    <Tooltip on='click' position='top' align='start'>Top Start</Tooltip>
    <Tooltip on='click' position='top' align='end'>Top End</Tooltip>
    <Tooltip on='click' position='right' align='start'>Right Start</Tooltip>
    <Tooltip on='click' position='right' align='end'>Right End</Tooltip>
    <Tooltip on='click' position='bottom' align='start'>Bottom Start</Tooltip>
    <Tooltip on='click' position='bottom' align='end'>Bottom End</Tooltip>
    <Tooltip on='click' position='left' align='start'>Left Start</Tooltip>
    <Tooltip on='click' position='left' align='end'>Left End</Tooltip>
  </Button>
</CodeExample>

<CodeExample desc='Tooltip adopts to content size by default'>
  <Button>
    <Text>Long Tooltip Text</Text>
    <Tooltip on={['click', 'hover', 'focus']}>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</Tooltip>
  </Button>
</CodeExample>

<CodeExample desc='Glass Theme (tooltip closes when the cursor is outside the element)'>
  <Button>
    <Text>Glass Effect</Text>
    <Tooltip theme='glass' on={['click', 'hover', 'focus']} position='bottom'>Tooltip</Tooltip>
  </Button>
</CodeExample>
